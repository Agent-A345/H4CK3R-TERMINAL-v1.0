## 🖥️ H4CK3R TERMINAL v1.0

A cinematic, interactive Fake Hacker Terminal Web App built using HTML, CSS, and JavaScript, simulating a real hacking environment with commands, animations, sound effects, and immersive UI experience.

## 🚀 Features
* **Command-Based Terminal Interface** – Execute commands like a real shell environment.
* **Boot Sequence Simulation** – Realistic startup logs with progress bar animation.
* **Command History Navigation** – Use arrow keys to access previous commands.
* **Auto-Suggestion System** – Intelligent command suggestions while typing.
* **Typing Animation Engine** – Outputs text with realistic typing effects.
* **Matrix Rain Effect** – Animated hacker-style background visuals.
* **Sound Effects System** – Keystroke, success, error, and hacking sounds.
* **Fake Hacking Simulation** – “hack system” command triggers full hacking sequence.
* **Progress Bar Animations** – Dynamic loading bars for hacking tasks.
* **Password Authentication System** – Simulated login with password prompts.
* **Root Mode Access** – Elevated privileges with special commands.
* **Multiple Built-in Commands** – Includes help, ls, cat, ping, nmap, ssh, etc.
* **Theme Switching** – Change terminal colors (green, amber, white).
* **Matrix Fullscreen Mode** – Enter immersive Matrix experience.
* **Glitch & CRT Effects** – Scanlines, flicker, and retro display effects.


2. **Open the project and navigate to the file**:
```

```
3. **Run the game in any modern browser**

## 🎮 How to Use
**Run Commands**
* Type commands in terminal
* Press Enter to execute

**Navigate History**
* Press ↑ / ↓ to scroll through previous commands

**Use Auto Suggestions**
* Start typing a command
* Select suggestions using arrow keys or click
